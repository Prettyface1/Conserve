# Conservation Fund

A Stacks blockchain project for conservation funding.

## Features
- Smart Contract in Clarity 4
- Next.js Web Frontend
- Stacks Wallet Integration
- Chainhooks Integration

## Project Structure
- `contracts/`: Clarity smart contracts
- `web/`: Next.js application
- `tests/`: Clarinet tests

## Setup
1. Install Clarinet
2. `clarinet check`
3. `cd web && npm install`
4. `npm run dev`
